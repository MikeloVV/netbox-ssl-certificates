How it works

This tool connects to the specified domain and port to retrieve the SSL/TLS certificate.

Examples
  • example.com on port 443
  • mail.example.com on port 993
  • 192.168.1.1 on port 8443
Note: The certificate will be automatically parsed and imported into NetBox.
Warning: Make sure the domain is reachable from the NetBox server.
CLI Alternative

You can also scan domains using the management command:

python manage.py scan_domain example.com

With options:

python manage.py scan_domain \
  example.com \
  --port 8443 \
  --name "My Certificate"
